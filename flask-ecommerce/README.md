## 📖 About

<div align="center">

<img src="https://img.shields.io/badge/E--Commerce-Backend-FF6B6B?style=for-the-badge&logo=shopify&logoColor=white" />

</div>

A **feature-rich E-Commerce backend** project built using **Flask** and **JSON file storage**. This lightweight yet powerful API provides complete product management, shopping cart functionality, order processing, and admin capabilities - perfect for learning or rapid prototyping! 🎯

### ✨ Why This Project?

- ✅ **Easy to understand** - Clean and simple code structure
- ✅ **No database required** - Uses JSON for data storage
- ✅ **RESTful API** - Industry-standard endpoints
- ✅ **Session management** - Built-in cart handling
- ✅ **Admin panel ready** - Product CRUD operations included
